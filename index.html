Qui suis-je ?

Salut ! Je suis Jérémy, un passionné du web toujours en quête de nouveaux défis. Que ce soit avec Symfony, JavaScript ou Flutter, j'adore construire des trucs de A à Z — que ce soit une petite appli perso ou un site plus sérieux. Quand je ne code pas, je fouine dans les perf, j'ajuste le responsive ou j'essaie de nouvelles librairies juste pour le fun. Bref, développeur le jour, bricoleur web la nuit.
